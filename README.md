# Social-Network-Api

Social network API

# Technologies

Node.js
Express
MongoDB
Mongoose

# Installation

1. Clone repo
2. npm i in terminal
3. have Mongo DB running in the background
4. npm start in terminal
5. Run localhost in Insomnia
